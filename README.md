## 📌 Sobre o curso

Este minicurso apresenta os fundamentos da linguagem **Python** aplicados à introdução ao **Machine Learning**, utilizando a IDE **Spyder** como ambiente de desenvolvimento.

O curso aborda:

- Conceitos básicos da linguagem Python  
- Organização de projetos e scripts  
- Uso da IDE Spyder  
- Estruturação e execução de códigos  
- Primeiros passos para aplicações em Machine Learning  

---

## 🖥️ Ambiente de Desenvolvimento

### Python
Python é a linguagem de programação utilizada no curso. É amplamente empregada em ciência de dados, estatística, inteligência artificial e machine learning.

### Spyder (IDE)
Spyder é um ambiente de desenvolvimento integrado (IDE) voltado para análise de dados científicos.

### Configuração sugerida da interface

- **Layout estilo RStudio:**  
  `Window → Layouts da janela → Layout Rstudio`

- **Alterar cores da interface:**  
  `Ferramentas → Preferências`

---

## 🧭 Estrutura da Interface do Spyder

A interface é dividida em quatro áreas principais:

1. **Editor de Script** – Histórico de códigos do projeto  
2. **Console** – Onde os comandos são executados  
3. **Variáveis e Plots** – Lista de objetos criados e gráficos  
4. **Help e Arquivos do Projeto** – Documentação e arquivos do projeto
